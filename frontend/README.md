# frontend/

This directory is reserved for the JavaScript/TypeScript frontend.

## Getting started (for the frontend developer)

The backend runs on **http://localhost:8000** by default.

Start it with:
```bash
cd ../backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Interactive API docs (auto-generated by FastAPI):
- Swagger UI → http://localhost:8000/docs
- ReDoc     → http://localhost:8000/redoc

Full API contract (request/response shapes + examples) → see `../API_CONTRACT.md`

## Base URL

```
http://localhost:8000
```

## Recommended frontend stack

Choose whatever you like. The backend is a plain JSON API — no coupling to any FE framework.

Suggested options:
- **React + Vite** (`npm create vite@latest frontend -- --template react-ts`)
- **Next.js** (`npx create-next-app@latest frontend`)
- **Plain HTML/JS** for rapid hackathon prototyping

## CORS

CORS is already enabled for all origins (`*`) during development.
No proxy configuration needed.
